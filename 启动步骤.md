# 旅游信息管理系统 - 启动步骤

## 一、环境准备

安装依赖：
```
pip install django mysqlclient pillow
```

## 二、数据库准备

1. 登录MySQL，创建数据库：
```sql
CREATE DATABASE travel_db CHARACTER SET utf8mb4;
```

2. 修改数据库密码（如需要）：
   编辑 travel_system/settings.py 第68行的 PASSWORD

## 三、初始化项目

依次执行以下命令：

```bash
# 1. 进入项目目录
cd /mnt/e/proj/zuoye/旅游/travel_system

# 2. 数据库迁移
python manage.py makemigrations
python manage.py migrate

# 3. 导入初始数据（景点、用户等）
mysql -u root -p travel_db < travel_db.sql

# 4. 启动服务
python manage.py runserver
```

## 四、访问系统

浏览器打开：http://127.0.0.1:8000/
